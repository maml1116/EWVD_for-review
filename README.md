# EWVD(Endangered Wildlife Visual Dataset)_for-review
The dataset comprises 7483 high-resolution images, encompassing 27194 instances of four endangered species (Amur tiger, giant panda, golden snub-nosed monkey, and Sichuan takin). Each image is annotated with bounding boxes and five critical attributes: bounding box size, target size category, number of targets, vegetation coverage, and occlusion level.The dataset was collected in Wolong National Nature Reserve and Northeast Tiger and Leopard National Park in China. All images were labeled in COCO format.

Samples from this repository are for preview purposes only and cannot be downloaded or used to train models without permission. The link to obtain all datasets is https://pan.quark.cn/s/8d0c8d3ddc4f. At present, the datasets are not open. If you need to follow up, please contact the author.
